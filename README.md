# HWTeXGenerator
Generates a TeX template by parsing CSE 311/312/446 homework assignments. 
1. Clone or download.
2. Run HWTeXGen.jar
3. Select class from the dropdown menu.
4. Browse the homework pdf e.g. hw0.pdf
5. Generate.
