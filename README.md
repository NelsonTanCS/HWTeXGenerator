# HWTeXGenerator
Generates a TeX template by parsing CSE 311/312/446 homework assignments. 
1. Download HWTeXGen.jar
2. Select class from the dropdown menu.
3. Browse your homework pdf.
4. Choose a save location (optional).
5. Generate.
